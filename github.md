# GitHub

## Initiate project

> cd _path/to/my_folder_<br>
> git init<br>
> git add .
> git commit -m "_Initial Commit_"<br>
> git remote add origin _lol.git_<br>
> git push -u origin master

## Change remote url

List your existing remote
> git remote -v

Modify remote
> git remote set-url origin lol.git

Check remote
> git remote -v
